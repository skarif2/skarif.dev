## Getting Started

```bash
# Clone the repo
$ git clone https://github.com/skarif2/skarif.dev.git

# Change directory to newly created one
$ cd skarif.dev

# Install dependencies
$  npm install

# Run server in develop mode
$  npm run develop
```

Now visit `localhost:8000`.
